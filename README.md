This app is designed to mimic a skincare and haircare agent. If the user enters their skin or hair concerns (the initial testing version only includes skin concerns), the agent will respond with suggestions.
Statelessness means that the agent does not automatically retain information between requests. This app handles that by creating a chat object in agent.py which adds on the user and assistant messages to its current information everytime it receives a message from either side. This makes sure that the agent has the full conversation and can reference previous information from the chat each time a new message is sent. Once the user enters 'quit', the conversation history is cleared.
